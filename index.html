Electron으로 데스크탑 런처를 개발하고, AWS S3와 연동하여 안정적인 게임 설치 및 자동 업데이트 기능을 구현했습니다.

AWS S3를 활용한 안정적인 게임 파일 배포

게임 클라이언트와 매니페스트 파일을 AWS S3에 업로드하고, 런처에서 AWS-SDK를 통해 안정적으로 파일을 다운로드하는 시스템 구축
